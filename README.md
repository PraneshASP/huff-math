<h1 align="center"> Huff Math </h1>

**Implementation of basic math operations using Huff**

## Setup

```sh
git clone https://github.com/PraneshASP/huff-math.git
cd huff-math
npm i
```

### Building and Testing

To compile your contracts, write:

```sh
npm run compile
```

To run unit tests:

```sh
npm run test
```

To run fuzz tests:

```sh
forge test
```
